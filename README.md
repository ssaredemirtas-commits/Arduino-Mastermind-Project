## ðŸ“– Project Description
This project implements the classic logic puzzle game **"Mastermind"** using an ATmega328P (Arduino Uno) microcontroller. The goal is to guess a secret 4-digit code generated by the system.

The project is developed in **C++** using **Visual Studio Code (PlatformIO)** and utilizes direct AVR register manipulation for efficient hardware control.

## ðŸŽ® Game Logic & Features
1. **Dynamic Difficulty:** The system reads a potentiometer value via ADC to determine the number of **"Lives"** (Attempts) before the game starts.
2. **Random Code Generation:** A random seed is generated using analog noise, ensuring a unique code every time.
3. **Feedback System:**
   - **EXACT:** Correct number in the correct position.
   - **NEAR:** Correct number but in the wrong position.
4. **Win/Loss States:**
   - **Victory:** LED animation plays when the code is cracked.
   - **Game Over:** If lives run out, the secret code is revealed on the LCD.
